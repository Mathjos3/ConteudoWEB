O Microsoft Access 2013 oferece uma incrível plataforma para o desenvolvimento de soluções de gerenciamento de banco de dados com ferramentas de personalização fáceis de usar. Caso nenhuma personalização por parte do usuário final seja necessária (incluindo modificações de relatório), você pode optar por distribuir essas soluções do Access 2013 para que elas sejam executadas sem a necessidade de uma instalação completa do Access 2013. Para isso, é preciso acondicionar e distribuir seu aplicativo com o Access 2013 Runtime.

O Access 2013 Runtime é similar aos runtimes anteriores no sentido de que todas as IUs relacionadas a design foram removidas ou desativadas.

Você não precisa adquirir nenhum produto especial para redistribuir o Access 2013 Runtime. Você pode redistribuir livremente ou direcionar os usuários para este download.

Aplicativos baseados na web para SharePoint que são criados com o Access 2013 não exigem um runtime - a única exigência é um navegador da web compatível.


Recursos do Access

Recursos Descrição
Vá além dos bancos de dados de área de trabalho O Access é muito mais que uma ferramenta de criação de banco de dados para área de trabalho. É possível usá-lo facilmente para criar aplicativos que ajudam a administrar uma empresa. Os dados do Access são automaticamente armazenados em vários bancos de dados na nuvem. Portanto, ele está muito mais seguro, e você pode compartilhar os aplicativos do Access com os colegas.
Crie aplicativos fáceis de personalizar Crie aplicativos adaptados à maneira pela qual você e seus clientes fazem negócios. Faça alterações nos aplicativos de forma fácil e rápida para atender às necessidades organizacionais em evolução.
Integre dados entre o Access e aplicativos de linha de negócios A biblioteca de conectores do Access oferece várias maneiras de integrar os dados de aplicativos e as fontes de dados que impulsionam os negócios. Cenários integrados em fontes de dados modernas geram informações e elementos visuais agregados na interface conhecida do Access.
Armazene os dados no SQL Armazene os dados no SQL Server e no Banco de Dados SQL do Microsoft Azure para proporcionar maior confiabilidade, segurança robusta, escalabilidade e capacidade de gerenciamento no longo prazo. Os aplicativos do Access aproveitam a sintaxe SQL padrão e um back-end crítico, implantados tanto localmente como na nuvem.
